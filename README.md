# InterviewApplicant

## 概要

面接希望者が入力した内容を管理するためのWebアプリケーション。

## 機能

**管理者側**

ログイン、質問内容の設定、回答結果の一覧表示、詳細表示、編集、削除

**応募者側**

回答の送信、画像や動画のアップロード

## 使用技術

Angular / TypeScript / SCSS / Firebase（Firestore, Authentication, Storage）
